# Aplicació de comandes entre clients i cuiners

Aquest repositori conté la documentació del projecte desenvolupat dins l'assignatura d'Enginyeria del Software.

## Descripció del projecte

El projecte consisteix en el desenvolupament d'una aplicació que permet als clients consultar plats preparats per cuiners, realitzar comandes i gestionar enviaments.
