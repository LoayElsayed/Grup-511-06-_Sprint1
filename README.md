# Aplicació de comandes entre clients i cuiners

Aquest repositori conté la documentació del projecte desenvolupat dins l'assignatura d'Enginyeria del Software.

## Descripció del projecte
El projecte consisteix en el desenvolupament d'una aplicació que permet als clients consultar plats preparats per cuiners, realitzar comandes i gestionar enviaments.

## Estructura del repositori

- **/documents/**
  Documents del projecte en format PDF i editable.

- **/requirements/**
  Fitxes individuals de tots els requisits funcionals i no funcionals.

- **/minutes/**
  Actes de les reunions de l'equip.

- **/others/**
  Documents addicionals o plantilles utilitzades durant el projecte.

## Sprint 1

Durant aquest sprint s'han definit:
- Requisits funcionals
- Requisits no funcionals
- Document d'especificacions
- Estructura inicial del repositori
